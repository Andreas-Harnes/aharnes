Java virtual Machine

Compiling code in Java is different then other languages. When the code is compiled it is turned into a file with a .class or .jar extension, and these files contain "bytecode". To run this code the JVM is needed.

The picture above shows the procedure from the .java file and untill the code is executed in Java Virtual Machine. The Virtual machine lies between the byte code and your Operating system. The byte code is executed on the virtual machine and the results are displayed on your computer. (Source)

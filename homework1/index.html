Welcome to this site about Java

Here we will talk about the programming language Java, and other subjects that is related to the programming language.

Java is OOP language that unlike c does not need to be compiled to machine code to be executed. The java project started in 1991 and was called Oak. Later it was renamed to Green and at last it was renamed to the name we know it by today, Java.

Inorder to run Java on your computer you need to install the JDK. In the JDK you will get the JRE which is needed to run your Java code. (Source)
